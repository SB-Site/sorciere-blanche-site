Politique de Confidentialité

Les Éditions de la Sorcière Blanche
Dernière mise à jour : 23 juillet 2025

Les Éditions de la Sorcière Blanche, maison d’édition associative bilingue français/anglais dédiée à la fantasy, l’ésotérisme et le bien-être, s’engagent à protéger vos données personnelles conformément au Règlement Général sur la Protection des Données (RGPD) et aux lois applicables. Cette Politique de Confidentialité explique comment nous collectons, utilisons, partageons et protégeons vos données personnelles via notre site web (sorciereblancheeditions.com, ci-après « le Site »).

1. Données collectées

Nous collectons les données suivantes :

  • Données d’identification : Nom, prénom, adresse email, adresse postale (pour les commandes physiques).
  • Données de paiement : Informations de paiement (ex. : numéro de carte via Shopify Payments, PayPal), sans stockage direct par nous.
  • Données de navigation : Adresse IP, type de navigateur, pages visitées, via des cookies (voir Politique de Cookies).
  • Données de contact : Messages envoyés via le formulaire de contact ou email.

2. Finalités du traitement

Vos données sont utilisées pour :

  • Traitement des commandes (ex. : achat de livres, livraison).
  • Gestion des abonnements de soutien (ex. : via PayPal).
  • Envoi de newsletters et offres promotionnelles (avec votre consentement).
  • Amélioration de l’expérience utilisateur (ex. : analyse via Google Analytics).
  • Réponse à vos demandes via le formulaire de contact.

3. Base légale

Le traitement de vos données repose sur :

  • Exécution du contrat : Pour traiter vos commandes et livraisons.
  • Consentement : Pour les newsletters et cookies non essentiels.
  • Intérêt légitime : Pour améliorer le Site et assurer sa sécurité.
  • Obligation légale : Pour répondre aux exigences fiscales ou judiciaires.

4. Partage des données

Vos données peuvent être partagées avec :

  • Partenaires de livraison : Gelato, Printful, Amazon KDP, IngramSpark (pour l’impression et l’expédition).
  • Partenaires de paiement : Shopify Payments, PayPal, Gumroad.
  • Partenaires marketing : Mailerlite (newsletters), Google Analytics (analyse anonyme).

Aucun transfert de données hors de l’Union européenne n’est effectué sans garanties appropriées (ex. : clauses contractuelles types).

5. Durée de conservation

Les données sont conservées pour :

  • Commandes : 5 ans après la dernière transaction (obligation fiscale).
  • Newsletters : Jusqu’à votre désinscription.
  • Cookies : 13 mois maximum (voir Politique de Cookies).
  • Messages de contact : 3 ans après la dernière interaction.

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Accès : Consulter vos données personnelles.
  • Rectification : Corriger des données inexactes.
  • Suppression : Demander la suppression de vos données.
  • Opposition : Vous opposer au traitement pour des raisons légitimes.
  • Portabilité : Recevoir vos données dans un format structuré.
  • Limitation : Restreindre le traitement dans certains cas.

Pour exercer vos droits, contactez-nous à contact@sorciereblancheeditions.com. Vous pouvez également introduire une réclamation auprès de la CNIL (www.cnil.fr).

7. Sécurité

Nous utilisons des mesures techniques et organisationnelles pour protéger vos données (ex. : cryptage SSL, accès restreint). Nos partenaires (Shopify, PayPal) respectent également des normes de sécurité élevées.

8. Modifications

Nous pouvons mettre à jour cette Politique de Confidentialité. Les modifications seront publiées sur le Site et prendront effet immédiatement. Votre utilisation continue du Site constitue une acceptation des modifications.

9. Contact

Pour toute question concernant cette politique, contactez-nous à :
Email : contact@sorciereblancheeditions.com
